# esp_code_generator
Das Projekt ist ein lokaler, webbasierter Code-Generator für ESP32 und andere Mikrocontroller. Über eine benutzerfreundliche WebUI (Gradio) kann der Nutzer entweder eine Funktionsbeschreibung eingeben oder eine .ino-Datei hochladen. Das System nutzt Ollama (oder optional Groq/Gemini) zur KI-gestützten Generierung modularer C++-Projekte mit .cpp/.h-Dateien, automatischer Bibliothekserkennung und korrektem platformio.ini. Es unterstützt 10+ Boards, aktiviert OTA nur bei Wi-Fi-fähigen Geräten und liefert ein ZIP mit README. Integrierte Kompilierung und Fehleranzeige ermöglichen sofortiges Feedback – ohne Dokumentation lesen zu müssen. Alles läuft lokal, sicher und offline.
